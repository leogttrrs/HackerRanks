Make queries about the tables "CITY" and "STATION"

The CITY table is described as follows:

- ID : NUMBER

- NAME: VARCHAR2(17)

- COUNTRYCODE: VARCHAR2(3)

- DISTRICT: VARCHAR2(20)

- POPULATION: NUMBER

The STATION table is described as follows:

- ID: NUMBER

- CITY: VARCHAR(21)

- STATE: VARCHAR2(2)

- LAT_N: NUMBER

- LONG_W: NUMBER
